# Base de Conocimiento Táctica (COVA-AI)

Coloque en esta carpeta (`/app/docs/`) todos los manuales, leyes o normativas que desee que la IA consulte.

**Formatos soportados:**
- Archivos `.txt`
- Archivos `.pdf`
- Archivos `.docx` (Word)

Cada vez que el Motor de IA (Python) se reinicie, leerá los archivos de esta carpeta y los indexará en memoria para responder preguntas.
